# 星夜营地 · 前端重设计规格

日期：2026-06-14
方向：**夜间编辑感的高级野兽派**（Neo-brutalist × Editorial，深色）

## 动机
现有界面「AI 味」重，来源：emoji 当图标、紫金渐变、处处发光 glow、统一大圆角卡片。本次重设计去除这些通用 AI 美学特征，确立一个有性格、可落地、与「星夜专注」气质连贯的设计语言。

## 设计语言
- 形状：2px 直角硬边（最多 2px 圆角），告别大圆角柔和卡。
- 阴影：硬投影 `4px 4px 0 #000`，零模糊；不再用发光。
- 填充：平涂大色块；唯一保留的渐变是冒险页「时辰天空」（氛围必需）。
- 层级：靠边框粗细 + 留白 + 单一主色，而非发光提亮。
- 图标：全部真实 SVG（见下）。

## 配色 Token（深色）
```
--bg #14120d  --surface #1d1a14  --surface-2 #262219  --line #3a3326
--text #ece3cf  --text-dim #9a907c
--accent #e0b24b（熔金，唯一主色）  --accent-hi #f0c862
--oxblood #b5462f（暗酒红：撤退/传说，克制使用）
稀有度：common 骨灰 / rare #7fa6c9 / epic #9a7bd0 / legendary 熔金
```

## 字体
- 大标题/页眉：衬线 `Songti SC / 思源宋体`（系统自带，零下载）；拉丁配 Fraunces（OFL，仅拉丁子集）。
- 标签/按钮/导航：系统重黑无衬线（可选自托管「得意黑」子集）。
- 数字：`tabular-nums` 等宽。
- 大写 + 字间距小标签做杂志感（如 `25 MIN · EXP 50`）。
- 刻意用系统 CJK 衬线，避免数 MB CJK 网络字体拖慢移动端。

## 图标系统（替换所有 emoji）
- 来源：game-icons.net（CC BY 3.0）做 RPG 内容图标；Phosphor（MIT）做 UI 图标。
- 落地：`Icon.jsx` + 精选 ~25 个内联 SVG，`currentColor` 主题化；README 加 CC-BY 署名。
- 范围：UI 图标 + 材料图标（按 key 映射）全换 SVG；生物图鉴 emoji 来自后端内容数据，本轮保留（放进精致 SVG 边框内），全换为后续可选项。

## 结构改造（每页）
- 顶栏：报头 masthead——方形硬边头像 + 衬线称号；资源 chip（边框 + SVG + 等宽数字）；EXP 改分段方块条。
- 营地：场景裹进硬边「取景窗」带标题条；委托改编辑式列表行（`01 — 今日委托` 大序号 + 衬线标题 + 细线分隔），hover 硬投影位移，熔金「出发 →」。
- 冒险：保留时辰天空；计时环改厚扁平无发光 + 等宽读秒；撤退用酒红 ghost。
- 图鉴：硬边卡，稀有度用角标 + 边框色；未解锁用斜线网纹替代灰度。
- 统计：编辑式数据版——大号等宽数字、硬边数据块、本周扁平粗柱 + 基线、学科带框轨道。
- 底栏：粗标签 + SVG 图标，激活态=熔金实心方块（滑动游标保留）。

## 动效与保留
- 动效改干脆利落（短、硬投影按压），去掉发光脉动；传说时刻改酒红/熔金编辑式接管取代 god-ray。
- 保留：prefers-reduced-motion 兜底、可访问 Modal、framer 分包、时辰天空、数字滚动、骨架屏、PWA、键盘焦点。
- 不动：后端 / API / 游戏逻辑 / 数据 / 测试。

## 实施
- 分支 `feat/redesign-brutalist`。
- 顺序：先建设计 token 系统（theme.css 重写）+ Icon 组件 + 基础组件类（锁定视觉语言以保证连贯），再按页重构 JSX（各 agent 仅改自身文件、只用既定 class），最后浏览器验证。
- 全程 build/test 保持绿；保留既有可访问性与性能优化。

## 范围边界（YAGNI）
- 不改服务端/数据/玩法。
- 不引入重型 CJK 网络字体。
- 生物 emoji→SVG 列为后续可选，不阻塞本次。
